Spring Security
se encarga de toda la parte de:
Autenticación → comprobar quién eres (login).
Autorización → verificar qué puedes hacer según tu rol (ADMIN, CLIENTE, etc.).
Protección → contra ataques como CSRF, inyección, sesiones inseguras, etc.

En resumen: Spring Security protege tu API y tus endpoints.
